# WirelessID 开发待办

> 说明：本清单基于当前工程目标整理。当前仍然**跳过了 `doc/Schematic.pdf` 的阅读**，但你已经补充了部分关键 GPIO 映射，因此本文已按最新硬件信息更新。

## 1. 总体目标

- 基于 **ESP32-C3 + ESP-IDF** 完成板级固件框架搭建。
- 支持以下核心功能：
  - CAN 通讯（TWAI）
  - 红外串口链路
  - ESP-NOW 设备组网
  - I2C 屏幕显示调试信息
  - WS2812B 状态灯驱动
  - UART0 串口收发
- 形成可长期维护的模块化工程结构，便于后续扩展协议、调试页面和设备管理逻辑。

#### UART0 当前卡点（2026-05）

- [x] 代码侧：`UART0 TX` 主动发送稳定（`U0_TX_ALIVE` 可观测）。
- [ ] 硬件侧：`UART0 RX` 回环未打通。

  - 待办：进行硬件隔离（单 TX 驱动 `U0RXD`）后复测。
